Bindingを持つViewにinitを実装するとき

ViewにBindingがあるときコンストラクタで何もしないのであれば基本的にはいい感じのinitが用意されるが、インスタンス化のタイミングで何かしたい場合はひと工夫必要

コンストラクタでBinding<>で受け取って_をつけてで代入するらしい PropertyWrappersの基本 参照

struct LoginView: View {
    @Binding var showingLogin:Bool

    init(showingLogin: Binding<Bool>) {
        self._showingLogin = showingLogin
        //do something
    }
}
