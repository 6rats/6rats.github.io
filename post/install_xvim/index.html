(入っていれば)古いのを削除

rm -rf ~/Library/Application\ Support/Developer/Shared/Xcode/Plug-ins/XVim2.xcplugin

codesign

https://github.com/XVimProject/XVim2/blob/master/SIGNING_Xcode.md

Keychainでの作業をやっているのであれば大抵下記のコマンド実行だけ

sudo codesign -f -s XcodeSigner /Applications/Xcode.app

clone & make

git clone git@github.com:XVimProject/XVim2.git XVim2
cd XVim2
make
