Installation

brew cask install bitbar
mkdir  ~/BitBarPlugins
cd ~/BitBarPlugins
curl https://raw.githubusercontent.com/matopeto/xcode-build-times/master/sources/xcodeBuildTimes.1m.php --output xcodeBuildTimes.1m.php
chmod +x xcodeBuildTimes.1m.php

Bitbarのpluginフォルダを~/BitBarPluginsに設定する

Xcode Preference

Behaviors → Edit Behaviors → Build → Starts,Succeeds,Failsの「Run」にxcodeBuildTimes.1m.phpを指定する
