ObservableObjectと@Published

import SwiftUI import Combine class Contact: ObservableObject { var name: String @Published var age: Int init(name: String, age: Int) { self.name = name self.age = age } func haveBirthday() -> Int { age += 1 return age } func changeName() { name = "Changed!" } } let john = Contact(name: "MyName", age: 10) var cancellable = john.objectWillChange .sink { _ in print(john.age) print(john.name) } john.haveBirthday() john.changeName() ageは@PublishedになっているのでobjectWillChangeは流れてくる nameは@PublishedつけてないのでobjectWillChangeは流れない

No More Google?

はじめに 先日ふとしたことから下記のようなページを見つけました No More Google 僕はGoogleのサービスをかなり利用しており、正直Googleなしでは生きていけないのでは？とすら思っていました。しかしあえてGoogleを使わない生活は可能なのか？置き換えるサービスはあるのか？単純な好奇心から実際に置き換えてみることにしました。前提として僕はGoogleが好きです。念のため。 事前の予測として、僕がサービスを選ぶ傾向としてアカウントあれこれ作るの面倒でGoogleに寄ってるのでまたどこかに寄っていくような気配はしています… 利用しているGoogleのサービス一覧 チェックが付いているのは現時点で置き換え完了 Google検索 Gmail YoutubeMusic Google Photo Google Chrome Google Drive Authenticator Google日本語入力 Calendar Google Analytics Google Adsense Google Map GooglePlayConsole(これは無理) 各サービスの置き換え Google検索 DuckDuckGoに置き換え 今のところ日常生活に支障なし Gmail iCloudメールに置き換え AppleIDにGmailを使っていたので適当なメールサービス(ProtonMailなど)で置き換えて、日常的に使うメールアドレスとしてはiCloudメールに切り替え。 メール自体をほぼ使わないで済むようにしている(メルマガや機械的な通知系は基本的にOFF)ので、メールの切り替え自体はそこまで大変じゃない。 色々なサービスのログインとかに使うからそれらの置き換えがちょっと面倒なくらい。気が付いたところからやればいい。 YoutubeMusic Apple Music or Spotify or その他 Google Play Musicの頃からファミリープラン(1480円/月)で使ってたけど、Youtube Musicに強制移行されるので見直すいいタイミング。 過去にGoogle Play Musicに上げた音楽の扱いが難しいと思ったけど、どのサブスクリプションサービスでも聴けるようなメジャーな音楽は自分で所有する必要はないので潔く破棄。マイナーな楽曲だけは手元にすべてダウンロードしておくという方針。 Google Takeoutで音楽を含めGoogleに預けてあるデータをすべてエクスポートできる。 あとはApple MusicかSpotifyかもしくは他のサービス（AWAとかLINE Musicとか）試していく感じ。 Google Photo iCloud Photoに置き換え 数年にわたってGoogle Photoに上げ続けてきたので、大変だろうと思いつつやってみた。 YoutubeMusic同様にGoogle Takeoutで写真はすべてエクスポートできる。Googleフォトでアルバムを作っている場合は写真が重複(原本の写真とアルバムにある写真は別物扱いされてる)してダウンロードされるので注意。僕はiCloud Photoに上げる前にDuplicate Photos Fixer Proというソフトウェアを使って重複を除去しました。

日本人の勝算―人口減少×高齢化×資本主義

読み途中だけど簡単にめも これからの日本の問題 人口は減っていって経済はものすごいスピードでシュリンクしていく 高齢化が加速度的に進むので人が減っていく上に消費が減る(年取るとそんなに消費しない) 解決策 生産性を上げる 企業を統廃合して効率を上げる(今は非生産的な中小企業が多すぎるからまとめて効率化する) 最低賃金を上げる(最低賃金のUPに対応できるように企業は効率を上げて頑張るはず、効率上げられない企業は残念ながら…) 賃金が上がれば消費も回る 最低賃金の管轄が経済産業省ではなく厚生労働省なのも問題、社会政策としてではなく経済政策とすべき 輸出をするべき(それにはある程度の企業規模が必要なのでなおさら企業は統合を進めて規模を大きくすべき) 個人的に思ったこと 労働力を搾取するような企業では働かないことも重要なのかも。賃金を上げろと言ってもなかなか会社は動かないから自ら動くのもいいのかもと思う。
