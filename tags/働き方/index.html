Xcodeでビルドに費やしてる時間を可視化する

Installation brew cask install bitbar mkdir ~/BitBarPlugins cd ~/BitBarPlugins curl https://raw.githubusercontent.com/matopeto/xcode-build-times/master/sources/xcodeBuildTimes.1m.php --output xcodeBuildTimes.1m.php chmod +x xcodeBuildTimes.1m.php Bitbarのpluginフォルダを~/BitBarPluginsに設定する Xcode Preference Behaviors → Edit Behaviors → Build → Starts,Succeeds,Failsの「Run」にxcodeBuildTimes.1m.phpを指定する Reference https://github.com/matopeto/xcode-build-times

リモートワークで気をつけていること

途中で職場が変わっているものの、2016年あたりから基本的にリモートで働いているので気をつけていることをメモ ※ この先も働く上で気がついたことは追記、更新していく予定ですが基本的には自分へのメモです ※ リモート関係ないこともあるかも ※ コロナウィルスの影響で同僚がほぼ全員リモートなので一部今の世の中に合ってないこともあるかも アウトプットは明確に 何をいつまでにやるのかは明確にしておく、事前に認識合わせしておく 例）〇〇機能を3月中に実装完了して4月頭にはテストを開始する 早めのアラート 仕事が間に合わないかもといった場合。オフィスで夜遅くまで残業していれば、誰かに気にしてもらえるがリモートだとそういう「雰囲気を察してもらう」ことはできないのでヤバそうなら早めに伝えておく タスクの棚卸し 前段のアラートは早めに上げる以前に「自分がやばい状態なのか気が付かない」ことがあるかもしれない そんなことがおきないようにタスクの棚卸しは高頻度でやる。終わってる？終わってない？何が揃えば終わり？終わってないけど大丈夫？ チームでチケット管理などをしていればそこにすべてを書き出せばいいし、ないのであれば手元で簡単なTODOリストを用意して整理整頓して管理 何か頼まれたりしたら、それがどんなに簡単なものであっても手を付ける前にまずはリストに追加して一旦リストの全体を見て何に手をつけたらいいのか判断 非同期を意識する 質問したり確認をお願いしたらレスが返って来るまでにある程度の時間がかかることを意識する。「待つ」ことはせずその間に別のタスクを進めてしまう 人によってレスのスピード違うのでそこはきちんと把握していく（即レスの人もいれば、1日経ってやっと返してくる人もいる） レスを「待つ」時間は細かいタスクを潰すいいタイミングだったりするので段取り大事 タスクの棚卸し、非同期を意識できるようになると少し自分で組み立てる努力をしてみる 例）〇〇機能の実装が終わって確認してもらおう。おそらく確認してレスをもらうのには1時間くらいはかかるだろうからサクッと終わりそうな細かいタスクをその間に2つ片付ける すぐに相談しないで調べる すぐに聞かない。まず30分くらい考えたり調べたりしていると問題への理解が深まって質問の精度をあげられることがある。あわよくば自己解決することもある わからないからといっていきなりチャットで質問すると説明に説明を重ねることになったりして相手と自分の時間を大幅に失う 電話しない 言った言わなかったになるから、なるべく電話しない。もし電話してもテキストでまとめて合意を取る なるべく即レス 即答できることは即答するが、即答できないときはちょっと調べますとか考えますとかまずは返しておく。長時間の無反応はなるべく避ける でも頑張りすぎると息が詰まるからほどほどに いつでも退職できる心構えでやる 本当に退職するかどうかはともかく、大抵の場合退職後はその業務にタッチできなくなるので、それなりの精度で引き継がないといけないはず。それくらいの気持ちでやるほうが業務の共有の精度があがる 単にドキュメントを残すだけでなく業務そのものを全く知らない第三者に引き継ぎやすいように普段から整備しておくのが重要 余計な推測はしない オフィスにいないのでオフィスで何が起こっているのかわからなくてモヤモヤすることがあるかもしれないが、推測はしない 身の回りはきれいにしておく 物理的にもデジタル的にも 物理的には掃除しておく、気が散るものを周りに置かない デジタル的には余計なメールやプッシュが飛んで来ないように、ふとしたときに開いてしまいそうな使ってないサービスは解約しておく 適度に運動する 腕立て伏せでもプランクでもいいので少しだけでもやる 頑張り過ぎない 姿が見えないからといって、過剰に仕事ぶりを見せようと頑張り過ぎない。 小さな気晴らしを取り入れる FLOWERで花を家に飾る https://flowr.is
